# NexArtWO — Work Order & Inspection Management System

Professional construction work order management system built for **R.C Art Construction LLC** (Oregon CCB #247277).

## 🏗️ Features

- **Dashboard** — Real-time overview of work orders, revenue pipeline, and activity feed
- **Work Orders** — Create, track, and manage construction work orders with status workflow
- **Service Library** — 36+ pre-built construction services (Electrical, Plumbing, HVAC, Roofing, etc.)
- **Client Management (CRM)** — Full client profiles with edit, payment terms, referral tracking
- **PDF Reports** — Professional estimates, proposals, invoices, and progress reports
- **Logo System** — Dual-logo circular cropper (app + documents)
- **Address Autocomplete** — Real-time address suggestions via OpenStreetMap
- **Cloud Database** — Supabase PostgreSQL with automatic seeding and localStorage fallback
- **Oregon CCB Compliance** — Built-in compliance tracking

## 🚀 Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS (no framework dependencies)
- **Database**: Supabase (PostgreSQL)
- **Hosting**: GitHub Pages
- **Icons**: Lucide Icons
- **Maps**: OpenStreetMap Nominatim
